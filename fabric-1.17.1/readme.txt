Installation: 

1. Run: fabric-installer-0.9.0.exe. 

2. Use the given defaults, and press 'install'.

3. Copy the folders 'mods' and 'resourcepacks'.

4. Go to: C:\Users\YOUR_USERNAME\AppData\Roaming\.minecraft

5. Paste both the 'mods' and 'resourcepacks' folders into the '.minecraft' folder.

6. Launch minecraft. You should see an option for 'fabric'. Use that.
